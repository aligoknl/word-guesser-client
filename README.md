# word-guesser-client

This project is the frontend of a word-guessing game application built with Vue.js. The game allows users to guess a 5-letter word within a limited number of attempts and time. This repository contains the code for the user interface and client-side logic.

## Features

- User registration with username validation
- Interactive word guessing game
- Leaderboard to display top scores

## Demo

Check out the live demo: [Live Demo](https://target-word.netlify.app/)

## Installation

To get started with the project, clone the repository and install the dependencies:

```bash
git clone https://github.com/aligoknl/word-guesser-client.git
cd word-guesser-client
npm install
npm run dev
```
